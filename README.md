# gin-memos

A REST API for user management and voice memos.

## Setup project

```bash
# claude code will help you setup everything
claude -p "/setup" --dangerously-skip-permissions
```

## Commands

| Resource      | Description               |
| ------------- | ------------------------- |
| `task dev`    | start app with hot reload |
| `task --list` | all available commands    |
